* About
    * 自由に使えるアドベンチャーゲーム制作ツールです
    * Windows/Mac/Linuxで動きます
    * 2D/3Dアクセラレーションを使わないため、環境を選ばず動きます

* Windows動作確認リスト
    * Windows XP 32bit
    * Windows 7 SP1 32bit
    * Windows 10 32bit

* Mac動作確認リスト
    * Mac OS X Snow Leopard 10.6.8
    * Mac OS X Lion 10.7.5
    * OS X Mavericks 10.9.5
    * OS X El Capitan 10.11.5

* Linux動作確認リスト
    * Ubuntu 16.04 64bit
